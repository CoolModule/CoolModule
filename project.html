Query 1:

How many characters not-white are employed in illegal professions?

How many characters not-white are employed in illegal professions?

SELECT ?character
WHERE {
    ?character cont:hasProfession ?profession .
    ?profession rdf:type cont:IllegalProfession .
MINUS {
    ?character cont:hasEthnicOrigin dataset:White .
}
} GROUP BY ?character

Query 3:

Which characters have professions somewhat related to drug?

SELECT ?character
WHERE {
    ?character cont:hasProfession ?profession .
    ?profession rdfs:label ?profName .
